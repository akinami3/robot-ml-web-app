üêõ Troubleshooting

Issue: Cannot connect to backend

Solution:

  • Make sure backend is running on http://localhost:8000
  • Check CORS settings in backend
  • Update js/config.js with correct backend URL

Issue: WebSocket connection failed

Solution:

  • Verify WebSocket URL in js/config.js
  • Check backend WebSocket endpoint is running
  • Look for CORS or firewall issues

Issue: External libraries not loading

Solution:

  • Check internet connection (libraries loaded from CDN)
  • Open browser console to see specific errors
  • Consider downloading libraries locally if needed
